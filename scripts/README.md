# Scripts

Utility scripts for build, deployment, and maintenance tasks.

## Available Scripts

### Development
- `seed-db.js` - Seed database with initial data
- `clear-cache.js` - Clear build cache

### Deployment
- `deploy.sh` - Deploy to production
- `build-prod.sh` - Build for production

### Maintenance
- `backup-db.js` - Backup database
- `optimize-images.js` - Optimize images

## Usage

```bash
# Run a script
node scripts/seed-db.js

# Or with bun
bun scripts/seed-db.js
```
