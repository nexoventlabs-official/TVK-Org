# Integration Tests

Place integration tests here for testing multiple components working together.

## Structure

```
integration/
├── api/
│   ├── images.test.ts
│   └── enquiries.test.ts
└── pages/
    ├── home.test.tsx
    └── gallery.test.tsx
```

## Running Tests

```bash
bun test:integration
# or
npm run test:integration
```
